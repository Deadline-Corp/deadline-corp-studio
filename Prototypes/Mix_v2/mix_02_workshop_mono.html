# manifesto.md last modified: 2026-05-08

We started this studio because we kept seeing the same thing: software that shipped late, over-budget, and built by people who'd never have to maintain it.

We build differently. Every engagement is founder-led. If you're working with us, you're working with Alex or Nikolay — not a project manager routing to a junior dev in another timezone.

"We believe simplicity is a technical decision. We believe documentation is part of the product. We believe honesty is the only project management tool that works."

Our stack choices are boring on purpose. next.js, postgres, python, n8n. Battle-tested, community-supported, yours to own when we're done. No proprietary black boxes.

We're based between Phuket and Bangkok. We work async-first. We take on three to four projects a quarter — not twenty. This is not scale. This is craft.
