# Free-Pilot («Founding Partners») — подключение клиента за ~30 минут

Модель: 3-5 дружественных бизнесов получают систему бесплатно (6 мес / навсегда
как первые) в обмен на: реальное использование, фидбек раз в неделю, кейс с
цифрами и отзыв. Каждый клиент — **отдельная копия** (свой Railway-проект, своя
БД, свои токены): физическая изоляция данных, white-label почти бесплатно.
Себестоимость: ~$5-10/мес Railway + копейки LLM.

## Подключение (техническая часть, ≈30 мин)

1. **Развернуть копию** (≈10 мин, из папки бота):
   ```bash
   ./deploy/new-client.sh barbershop "Барбершоп Усы"
   ```
   Скрипт: новый Railway-проект + Postgres + уникальные токены + деплой + домен,
   в конце печатает токен владельца и оставшиеся шаги.
2. **LLM-ключи**: `railway variables --set OPENROUTER_API_KEY=...`
   (+ `GROQ_API_KEY` для распознавания голосовых).
3. **Telegram-бот клиента** (5 мин): @BotFather → `/newbot` (имя под бренд клиента)
   → токен и чат уведомлений в variables → `setWebhook` на
   `<домен>/webhooks/telegram` с `secret_token` из шага 1.
4. **Виджет на сайт клиента**: перед подключением скрипта задать
   `window.DEADLINE_BOT_API="<домен-копии>/chat"`; домен сайта клиента —
   в `ALLOWED_ORIGINS`.
5. **Передать клиенту**: ссылку `<домен>/admin/ui/` + токен владельца (лично,
   не в открытый чат). **Дальше он сам**: онбординг спросит название бизнеса,
   нишу (пресет), предложит демо-данные и проведёт обучение. Менеджерам он сам
   выдаёт токены в Настройках → Команда.
6. Записать project_id в `deploy/clients.txt`.

## Эксплуатация

- **Обновления всем сразу**: `./deploy/update-all.sh` (деплоит текущую ветку во
  все копии из clients.txt; миграции идемпотентны). Сначала обновляем СЕБЯ,
  день полёт нормальный → раскатываем клиентам.
- **Мониторинг**: каждому проекту UptimeRobot на `/health` (бесплатный план).
- **Поддержка**: общий TG-чат с клиентом; его утренний дайджест работает из
  коробки — система сама напоминает о себе.
- **Деньги потом**: дайджест/аналитика копят доказательства ценности
  («N лидов, M созвонов за месяц») — это и есть аргумент при переходе на платный.

## Правила пилота (проговорить словами при подключении)

- Бесплатно = взамен фидбек (15 мин раз в неделю) + кейс с цифрами + отзыв.
- Мы можем смотреть их панель (admin-доступ остаётся у нас) — для поддержки.
- LLM-расходы наши, пока разумно (< ~$10/мес на клиента); всплеск — обсуждаем.
- Прекратить пилот можно в любой момент с обеих сторон; данные отдаём (CSV).
