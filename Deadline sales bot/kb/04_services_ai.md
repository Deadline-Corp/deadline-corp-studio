# Услуга 03 — AI-агенты / AI Agents

## RU

Что делаем:
- Чат-боты, голосовые ассистенты, RAG-системы
- Автономные агенты — не игрушки и не demo
- Трекинг затрат и качества каждого ответа
- Эскалация на человека там, где модель неуверенна
- Versioning prompts — откатишься на любую версию
- A/B-тесты моделей на реальных данных клиента
- Multi-language из коробки — RU, EN и далее по списку

Девиз направления:
> Не chatbot. Production-assistant с метриками.

Типичные задачи:
- Чат-бот на сайт / приложение / в Telegram-канал
- AI-консьерж / first-line support
- RAG над документацией компании / KB
- Voice-assistant для call-центра
- Внутренние агенты для команды (Slack/Teams боты)
- Классификация / маршрутизация заявок
- Генерация контента под брендовый ToV

Стек (примеры):
- OpenAI / Anthropic / GLM / DeepSeek / open-source через OpenRouter
- LangChain / LlamaIndex / LangGraph
- Векторные БД: pgvector, Qdrant, Chroma
- Eval: RAGAS / DeepEval / Phoenix
- Observability: LangSmith / Arize

Подходит для запросов:
- Хочу AI-чат-бота на свой сайт
- Автоматизировать поддержку
- RAG / база знаний с поиском
- Voice-бот / голосовой ассистент
- Внутренний AI-агент для команды
- Замена / усиление call-центра

## EN

What we build:
- Chatbots, voice assistants, RAG systems
- Autonomous agents — not toys, not demos
- Cost and quality tracking on every response
- Escalation to a human where the model is unsure
- Prompt versioning — roll back to any version
- A/B tests of models on the client's real data
- Multi-language out of the box — EN, RU, and the rest

Department motto:
> Not a chatbot. A production assistant with metrics.

Typical tasks:
- Site / app / Telegram chatbot
- AI concierge / first-line support
- RAG over company docs / KB
- Voice assistant for call center
- Internal team agents (Slack/Teams bots)
- Ticket classification / routing
- Content generation in brand ToV

Stack (examples):
- OpenAI / Anthropic / GLM / DeepSeek / open-source via OpenRouter
- LangChain / LlamaIndex / LangGraph
- Vector DBs: pgvector, Qdrant, Chroma
- Eval: RAGAS / DeepEval / Phoenix
- Observability: LangSmith / Arize

Match for these requests:
- Want an AI chatbot on my site
- Automate support
- RAG / searchable knowledge base
- Voice bot / voice assistant
- Internal AI agent for the team
- Replace / augment call center
